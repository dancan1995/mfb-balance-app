# ClinCAT - Balance Assessment System

A Computer Adaptive Testing (CAT) application for clinical balance assessment research, featuring BERG Balance Scale and Functional Gait Assessment (FGA) with intelligent item selection.

## 🎯 Project Overview

ClinCAT is designed to revolutionize balance assessment in clinical settings by:

- **Reducing administration time** through adaptive testing
- **Collecting precise timing data** for research purposes
- **Building comprehensive item banks** for CAT development
- **Supporting multiple assessment types** (BERG, FGA, Adaptive CAT)

## 🚀 Quick Start

### Prerequisites

- Node.js 16+ 
- npm or yarn
- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)

### Installation

```bash
# Clone or create the project directory
mkdir ClinCAT && cd ClinCAT

# Initialize and install dependencies
npm install

# Start development server
npm start
```

The application will open at `http://localhost:3000`

### For Expo Development

```bash
# Install Expo CLI if not already installed
npm install -g @expo/cli

# Start Expo development server
npx expo start
```
